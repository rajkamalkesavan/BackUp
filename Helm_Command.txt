Show and inspect:
------------------

------------------

--set-string

Get Command:
-------------
helm get

helm ls -A  -> whether release admin installed or not

helm get manifest [RELEASE NAME] --namespace=[NAMESAPCE NAME]  --> It will show all the YAML files.

helm get values [RELEASE NAME] --namespace=[NAMESAPCE NAME]

helm get hooks [RELEASE NAME] --namespace=[NAMESAPCE NAME]

helm get all [RELEASE NAME] --namespace=[NAMESAPCE NAME]  --> will show all about release.




Uninstall Command:
------------------

helm uninstall --help

helm uninstall [RELEASE NAME]

helm uninstall [RELEASE NAME] --keep-history  --> it will maintain in history but it will not show in Release name list

	helm status [RELEASE NAME]

	helm rollback [RELEASE NAME]  --> it will reappear from history.
	
	helm status [RELEASE NAME]

Upgrade Command part -2:
------------------------

helm install [RELEASE NAME] [CHART NAME] --version 9.2.3 --set wordpressBlogName="happy helming blog"  --> it will change rivision as well

helm get values [RELEASE NAME]

helm upgrade [RELEASE NAME] [CHART NAME] --version 9.2.3 --set wordpressBlogName="happy helming blog"  --> it will change rivision as well


helm get values [RELEASE NAME] --> we can see only updated values
helm get all [RELEASE NAME]


helm upgrade [RELEASE NAME] [CHART NAME] --version 9.2.3 --set wordpressBlogName="happy to change" --reuse-values --> it will show old values and new values which we changed recently.


helm upgrade [RELEASE NAME] [CHART NAME] --reset-values --> it will revert all our custom changes and it will keep as it is in Docker image/WAR


helm upgrade [RELEASE NAME] [CHART NAME] -i -> i means if mentioned release is not there it will install.



Helm History Command:
---------------------

helm history --help

helm history [RELEASE NAME]  --> it will show all details of releases like revision and revision details as well.


Rollback command:
------------------

helm rollback --help

helm history [RELEASE NAME]  --> it will show all details of releases like revision and revision details as well.

helm rollback [RELEASE NAME] [REVISION NUMBER]

helm history [RELEASE NAME]



Pull command:
-------------

helm pull --help


helm pull [CHART NAME]  -d C:\temp\helmpackages --> it will download in to my local system.

helm pull [CHART NAME] -d C:\temp\helmpackages --untar --> it will download along with extracted tar


Template Command:
-----------------

helm template --help

helm template [RELEASE NAME] [CHART NAME]   --> it will rendered along with latest values.


helm install [RELEASE NAME] [CHART NAME]  --dry-run --debug --> it will install in to kubernetes cluster and it will show all the parameters and Notes details.
